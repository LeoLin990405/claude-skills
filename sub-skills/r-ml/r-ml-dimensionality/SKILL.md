# Dimensionality Reduction

Reduce data dimensions while preserving structure.

## Core Packages

| Package | Purpose |
|---------|---------|
| stats | PCA (prcomp, princomp) |
| Rtsne | t-SNE |
| umap | UMAP |
| irlba | Fast SVD/PCA |
