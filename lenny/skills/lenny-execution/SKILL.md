# Lenny Skills: Execution

3 actionable skills for shipping products.

### 1. Shipping Products
- Set clear launch criteria
- Cut scope to hit dates
- Celebrate launches

### 2. Managing Timelines
- Break work into milestones
- Build in buffer time
- Communicate changes early

### 3. Cross-functional Collaboration
- Build relationships proactively
- Understand other teams' priorities
- Over-communicate status
