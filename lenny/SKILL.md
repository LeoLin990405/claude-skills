## Playbooks (Role-Based)

For role-specific guidance, see [pm-playbooks](skills/pm-playbooks/SKILL.md):

1. **Startup Founder** → discovery → strategy → execution → growth → team → leadership
2. **First-Time Manager** → team → communication → execution → leadership
3. **Product Manager** → discovery → strategy → execution → communication → leadership
4. **Growth Leader** → growth → discovery → analytics → communication → leadership
5. **Engineering Manager** → execution → analytics → team → communication
6. **Sales Leader** → team → communication → strategy → growth
7. **AI Builder** → leadership → strategy → discovery → growth → execution
8. **Executive** → communication → leadership → strategy → execution
9. **Career Growth** → leadership → communication → team
