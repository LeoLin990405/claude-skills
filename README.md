# Claude Design Skills

Design and creative toolkit for Claude Code - algorithmic art, visual design, theming, frontend design, and web artifacts.

## Overview

This skill collection provides creative and design capabilities, from generative art to frontend UI design.

## Skills Included

| Skill | Description |
|-------|-------------|
| `algorithmic-art` | p5.js generative art with seeded randomness |
| `canvas-design` | Visual art design in PNG/PDF |
| `theme-factory` | Theme styling toolkit for artifacts |
| `frontend-design` | Frontend interface design |
| `web-artifacts-builder` | Web artifacts construction |
| `slack-gif-creator` | Slack GIF creation |
| `brand-guidelines` | Anthropic brand guidelines |

## Installation

```bash
cd ~/.claude/skills
git clone https://github.com/LeoLin990405/claude-design-skills.git
```

## Usage

```
/algorithmic-art       # Create generative art
/canvas-design         # Design visual art
/theme-factory         # Apply themes to artifacts
/frontend-design       # Design UI components
```

## Features

- **Algorithmic Art**: Flow fields, particle systems, interactive parameters
- **Canvas Design**: Posters, visual compositions, design philosophy
- **Theme Factory**: 10 preset themes, custom theme generation
- **Frontend Design**: UI components, layouts, responsive design
