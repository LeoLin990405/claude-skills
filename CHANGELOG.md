# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-03-16

### Added

- **PM Toolkit** (`lenny/`) -- 88 skills across 9 product management domains with 10 templates
- **Design Toolkit** (`design/`) -- 11 skills for generative art, visual design, frontend, theming, and design systems
- **Document Toolkit** (`office/`) -- 5 skills for PDF, Word, Excel, PowerPoint with cross-format workflows
- **R Analytics Toolkit** (`r-analytics/`) -- 16 sub-skill domains covering 250+ R packages
- **Knowledge Toolkit** (`obsidian/`) -- 3 skills for Obsidian Markdown, Bases, and JSON Canvas
- **Repository Toolkit** (`github-repo-design/`) -- 15 modules for GitHub repo structure, CI/CD, security, and versioning
- **Developer Toolkit** (`utility/`) -- 6 skills for skill creation, MCP building, docs, testing, and communications
- **Coordination Toolkit** (`ccb/`) -- 14 skills for multi-AI delegation and orchestration
- Cross-toolkit workflow compositions
- Repository meta files: README, LICENSE, CONTRIBUTING, CHANGELOG
- GitHub issue templates and PR template
- Claude Code review workflow (`.github/workflows/claude-review.yml`)
