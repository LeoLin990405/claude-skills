# Pend Provider

Fetch the latest replies for a provider via the unified `pend` command.

## Execution (MANDATORY)

```
Bash(pend $ARGUMENTS)
```

## Rules

- Arguments must start with the provider name (e.g., `gemini`).
- Only use when the user explicitly requests results.
